# 屏幕预警for EVEonline — 使用说明

## 概述

屏幕预警 是一款 macOS 屏幕颜色监测工具。在屏幕任意区域持续监测指定颜色，一旦检测到目标颜色即触发声音警报 + 系统通知。适用于游戏（如 EVE Online）中特定 UI 元素变化预警、直播状态监控等场景。

---

## 一、基本操作流程

### 1. 选择监测区域
点击任务卡片上的 **「选区」** 按钮，屏幕进入选区模式：
- 鼠标拖拽框选要监测的屏幕区域
- 选区背景会显示半透明屏幕截图，框内区域清晰可见
- 按下 **ESC** 或框选过小区域可取消

### 2. 设置目标颜色
在右侧设置区点击 **「目标颜色」** 旁的色块打开系统取色器，选择要检测的目标颜色。

### 3. 开始监测
点击 **「开始监测」** 按钮，应用将以每 0.5 秒的频率截取所选区域并与目标颜色比对。

---

## 二、任务管理

支持同时监测多个区域，每个任务独立配置。

| 操作 | 方法 |
|------|------|
| 新增任务 | 点击任务列表右上角 **＋** 按钮 |
| 删除任务 | 点击任务行右侧 **🗑** 按钮（至少保留 1 个） |
| 启用/禁用 | 点击任务行左侧开关 |
| 选择编辑任务 | 点击任务卡片，高亮边框表示当前编辑中的任务 |

> 下方设置区（颜色容差、匹配占比等）始终作用于**当前高亮选中的任务**。

---

## 三、检测参数说明

### 颜色容差
范围：`0.01 — 0.50`（默认 0.08）

每个 RGB 通道允许的偏差范围。值越小，颜色匹配越严格。调高可兼容屏幕颜色轻微波动。

### 匹配占比
范围：`1% — 30%`（默认 5%）

选区中匹配像素占总采样像素的比例阈值。达到该比例即判定为「检测到颜色」。适合有噪声背景的场景适当调高。

### 采样密度
| 档位 | 步长 | 适用场景 |
|------|------|----------|
| 高 | 1px | 极小目标区域，需精确检测 |
| 中 | 4px | 常规区域（默认） |
| 低 | 8px | 大面积区域，降低 CPU 占用 |

### 匹配策略
| 策略 | 原理 | 适用场景 |
|------|------|----------|
| 逐通道 | R、G、B 各自独立比对 | 精确颜色匹配 |
| 欧几里得 | 颜色空间距离比对 | 对渐变色/光照变化容忍度高 |

> 切换策略后需重新设置合适的「颜色容差」。欧几里得模式下容差建议在 0.15—0.35 之间。

---

## 四、警报策略

### 连续匹配
选项：`1次 / 2次 / 3次 / 5次`（默认 2 次）

目标颜色需连续 N 帧被检测到，才触发警报。防止短暂误触（如画面闪烁）。

### 警报冷却
选项：`0.5s / 1.5s / 3s / 5s`（默认 1.5s）

两次警报之间的最小间隔。避免颜色持续存在时警报连发。

---

## 五、通知与音频

### 系统通知
开关控制是否弹出 macOS 系统通知。首次开启时系统会请求通知权限。

### 警报音量
范围：`0% — 100%`（默认 80%）

### 自定义音频
点击 **「选择…」** 可替换默认嘟声为任意音频文件（支持 .mp3 / .wav / .m4a 等）。点击 **✕** 恢复默认。

### 测试警报
点击 **「测试警报」** 按钮立即播放一声，用于确认音量和音频文件设置正确。

---

## 六、监测过程中的界面

- 每个活跃任务有独立**彩色浮窗**标记监测区域（红色圆角边框 + 四角加粗）
- 浮窗可**拖动**调整监测位置，关闭后位置不保存
- 下方**实时截图**区显示各任务最新截取的预览图
- 状态指示灯：
  - 🔴 红色闪烁 = 检测到目标颜色
  - 🟢 绿色常亮 = 监测中，未匹配
  - ⚪ 灰色 = 待机
  - 🔵 蓝色 = 选区模式中

## 七、常见问题

**Q: 启动后提示「请先选择监测区域」？**
A: 需要为至少一个启用的任务设置监测区域（点击「选区」按钮框选）。

**Q: 检测不到颜色？**
A: 尝试增大「颜色容差」或降低「匹配占比」。确认截图预览显示的区域是否正确。

**Q: 屏幕录制权限？**
A: 首次截图时系统会弹出权限请求。如误拒，前往「系统设置 → 隐私与安全性 → 屏幕录制」手动授权。

**Q: 通知不弹？**
A: 确认「系统通知」开关已开启。如从未弹过权限请求，关闭再开启一次开关即可触发。

**Q: 程序坞找不到应用？**
A: 应用图标在程序坞中可见。如窗口关闭，通过 Spotlight（⌘+空格 搜索「屏幕预警」）重新打开。

---

## 八、配置持久化

所有设置（任务、颜色、容差、音量等）自动保存在：
```
~/Library/Application Support/ScreenColorAlert/config.json
```
卸载时删除此目录即可清除所有配置。日志文件在同一目录的 `Logs/` 子文件夹中。
